How It Works

  • Downloads and saves HTML from the previous day's box scores.
  • Merges the data into a single AI prompt.
  • Sends the prompt to an AI model to generate the transcript.
  • Converts the transcript into an MP3 using TTS.
  • Uploads assets to Amazon S3.
  • Updates and publishes the RSS feed.

Currently this is a manual process while I monitor cost and stability. Automation via cron job coming soon!
